Agreement Verify Rule
TPSZ DUP ERR
ALL and individual TPSZ can’t be inserted in same row.
(e.g. AL2 and D2 can’t be updated in same row but separate rows.)
EFFECTIVE DATE ERR
From date must be earlier than "To date".
CUST ERR
Customer code is not registered in MDM.
CMDT GRP ERR
Commodity group code is not registered in MDM.
FM NODE ERR
From node is not registered in MDM yard or MDM leasing company yard.
VIA NODE ERR
Via node is not registered in MDM yard.
DOOR NODE ERR
Door node is not registered in MDM zone.
TO NODE ERR
To node is not registered in MDM yard or MDM leasing company yard.
REVERSE  FLAG ERR
The route pair which contains "Via node" is not compatible with reverse function.
Please make reverse flag "N" to update this route pair.
WGT INPUT CHK
Weight info. must be input for "Full CNTR" rows.
LINE  + ROW_NO + DUP

Same info. exists in the database. (This is not an error alert message.)

Users can proceed uploading without any problems.
