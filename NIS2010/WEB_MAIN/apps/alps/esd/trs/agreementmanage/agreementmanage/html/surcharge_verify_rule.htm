Agreement Verify Rule (Surcharge)
TPSZ DUP ERR
ALL and individual TPSZ can’t be inserted in same row.
(e.g. AL2 and D2 can’t be updated in same row but separate rows.)
EF DATE ER
From date must be earlier than "To date".
Effective date must conform to forms as YYYY-MM-DD.
CUST ERR
Customer code is not registered in MDM.
CMDT GRP ERR
Commodity group code is not registered in MDM.
FM NODE ERR
From node is not registered in MDM yard or MDM leasing company yard.
VIA NODE ERR
Via node is not registered in MDM yard.
DOOR NODE ERR
Door node is not registered in MDM zone.
TO NODE ERR
To node is not registered in MDM yard or MDM leasing company yard.
SCG ERR
Surcharge code must be input.
SCG DIV ERR
Fixed Ratio Div must be input.
ALL ROUTE FLAG ERR
All route flag & route information can’t be inserted at the same time.  
1) FUA - All route flag must be "Y" 
2) FUE - All route flag can't be used. Route information must be inserted.
3) VAT
   a) For all route application, all route flag must be "Y" without route information.
   b) For certain route application, all route flag must be "null or N" with route information.

DUP
Same info. exists in the database. (This is not an error alert message.)

Users can proceed uploading without any problems.
